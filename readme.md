# PaperForge

**論文を自分の知識資産に変える** パーソナルナレッジエージェント

## 🎯 コンセプト

論文を読んで終わりにしない。  
概念と関係性を抽出し、自分だけのナレッジグラフを構築。  
エージェントと対話しながら、知識を育てる。

## 🛠️ 技術スタック

| レイヤー | 技術 | 役割 |
|----------|------|------|
| 概念抽出 | Gemma 2 + LoRA (Cloud GPU) | 論文特化ファインチューニング |
| 対話・推論 | Gemini API (Vertex AI) | エージェントの思考 |
| エージェント | ADK | マルチエージェント構成 |
| 類似検索 | Vertex AI Vector Search | 概念間の関連発見 |
| 永続化 | Firestore | グラフデータ保存 |
| デプロイ | Cloud Run | UI + API |
